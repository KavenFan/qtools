# qtools
一些写代码时用到的极简实用工具

目前支持或计划支持以下功能:

1. 模板生成器。输入模板和数据批量生成结果。
2. proto定义自动编号。将proto定义自动从1开始编号。
3. 列表markdown转proto定义。将markdown文档的入参出参定义转成proto定义。
4. json与url互转。json格式和url格式的参数进行转换。
